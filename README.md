Wordlish Arcade
This is a 5-letter word guessing game built with React and TypeScript.

Features
Classic Wordle gameplay with 6 attempts to find the secret word.
A live leaderboard that pulls player scores from a data service.
Responsive design that looks good on both desktop and mobile.
Loading screens that show "Preparing your word..." while data is fetching.

Features: All the logic for the game and the leaderboard is kept in its own separate folder.

Pages: Each screen (Home, Play, Leaderboard) is a self-contained unit.

Shared: Common tools, like the custom data-fetching hook, are kept in a shared folder for everyone to use.
